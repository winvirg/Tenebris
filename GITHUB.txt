git clone (link) : Clona repositorio remoto na maquina local;

git status : Mostra o nome da branch atual, se ela está atualizada com a branch remota, se há alterações na não indexadas(arquivos em vermelho) e se existem alterações indexadas(arquivos de verde) que ainda não foram commitadas;

git log : Mostra o log da branch local atual;

git pull orgin <branch> : Puxa as alterações da branch referenciada(modificações);

git add . : Adiciona todas as alterações para a staging (indexa alterações dos arquivos); 

git add <nome arquivo>.. : Adiciona as alterações de um <arquivo> definido para a staging (indexa alterações dos arquivos); 

git commit -m <mensagem> : Faz um commit com as alterações indexadas com a <mensagem> definida;

git push origin <branch-remota> : Empurra da branch local para <branch-remota>;

git checkout <branch> : Muda para <branch> definida;

git checkout -b <branch> : Cria uma nova <branch> local a partir do ponto atual;

git branch -D <branch> : Apaga uma <branch> local definida;

git restore (pasta ou arquivo) : Restaura uma pasta ou arquivo modificado/excluido

git restore . : Restaura todos os arquivos

git reset <codigo-commit> --hard : Volta o repositório para o estado do commit definido
